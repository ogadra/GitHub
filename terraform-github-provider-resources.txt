Terraform GitHub Provider (integrations/github) で管理できるもの
=================================================================

ソースコード (provider.go) に基づく完全なリストです。
参照: https://github.com/integrations/terraform-provider-github/blob/main/github/provider.go


=====================================
リソース (Resources) — 作成・変更・削除が可能
=====================================

【リポジトリ管理】
  github_repository                          - リポジトリの作成・管理
  github_repository_file                     - リポジトリ内のファイル管理
  github_repository_topics                   - リポジトリのトピック（タグ）管理
  github_repository_collaborator             - コラボレーター（個別）の管理
  github_repository_collaborators            - コラボレーター（一括）の管理
  github_repository_deploy_key               - デプロイキーの管理
  github_repository_webhook                  - Webhookの管理
  github_repository_project                  - リポジトリプロジェクトの管理
  github_repository_milestone                - マイルストーンの管理
  github_repository_pull_request             - プルリクエストの管理
  github_repository_autolink_reference       - オートリンク参照の管理
  github_repository_custom_property          - リポジトリのカスタムプロパティ
  github_repository_dependabot_security_updates - Dependabotセキュリティアップデート設定

【ブランチ管理】
  github_branch                              - ブランチの作成・管理
  github_branch_default                      - デフォルトブランチの設定
  github_branch_protection                   - ブランチ保護ルール（GraphQL API）
  github_branch_protection_v3                - ブランチ保護ルール（REST API v3）

【リポジトリ環境 (Environments)】
  github_repository_environment              - デプロイ環境の管理
  github_repository_environment_deployment_policy - 環境デプロイポリシーの管理
  github_repository_deployment_branch_policy - デプロイメントブランチポリシーの管理

【ルールセット】
  github_repository_ruleset                  - リポジトリレベルのルールセット
  github_organization_ruleset                - Organizationレベルのルールセット

【GitHub Actions】
  github_actions_secret                      - リポジトリレベルのシークレット
  github_actions_variable                    - リポジトリレベルの変数
  github_actions_environment_secret          - 環境レベルのシークレット
  github_actions_environment_variable        - 環境レベルの変数
  github_actions_organization_secret         - Organizationレベルのシークレット
  github_actions_organization_secret_repositories  - Orgシークレットのリポジトリアクセス（一括）
  github_actions_organization_secret_repository    - Orgシークレットのリポジトリアクセス（個別）
  github_actions_organization_variable       - Organizationレベルの変数
  github_actions_organization_variable_repositories - Org変数のリポジトリアクセス（一括）
  github_actions_organization_variable_repository   - Org変数のリポジトリアクセス（個別）
  github_actions_organization_permissions    - OrganizationレベルのActions権限
  github_actions_organization_workflow_permissions  - Organizationワークフロー権限
  github_actions_repository_access_level     - リポジトリのActionsアクセスレベル
  github_actions_repository_permissions      - リポジトリのActions権限
  github_actions_runner_group                - ランナーグループの管理
  github_actions_hosted_runner               - ホステッドランナーの管理
  github_actions_organization_oidc_subject_claim_customization_template - Org OIDCサブジェクトクレームのカスタマイズ
  github_actions_repository_oidc_subject_claim_customization_template   - リポジトリOIDCサブジェクトクレームのカスタマイズ
  github_workflow_repository_permissions     - ワークフローリポジトリ権限

【チーム管理】
  github_team                                - チームの作成・管理
  github_team_membership                     - チームメンバーシップ（個別）
  github_team_members                        - チームメンバー（一括）
  github_team_repository                     - チームへのリポジトリアクセス付与
  github_team_settings                       - チーム設定の管理
  github_team_sync_group_mapping             - チームIdP同期グループマッピング

【Organization管理】
  github_membership                          - Organizationメンバーシップの管理
  github_organization_settings               - Organization設定の管理
  github_organization_block                  - ユーザーのブロック
  github_organization_project                - Organizationプロジェクトの管理
  github_organization_webhook                - Organization Webhookの管理
  github_organization_security_manager       - セキュリティマネージャーチームの管理
  github_organization_custom_role            - カスタムロールの管理
  github_organization_custom_properties      - Organizationカスタムプロパティ
  github_organization_repository_role        - リポジトリロールの管理
  github_organization_role                   - Organizationロールの管理
  github_organization_role_team              - ロールへのチーム割り当て
  github_organization_role_user              - ロールへのユーザー割り当て
  github_organization_role_team_assignment   - ロールチーム割り当て

【Dependabot】
  github_dependabot_secret                   - リポジトリレベルのDependabotシークレット
  github_dependabot_organization_secret      - OrganizationレベルのDependabotシークレット
  github_dependabot_organization_secret_repositories - Org Dependabotシークレットのリポジトリアクセス（一括）
  github_dependabot_organization_secret_repository   - Org Dependabotシークレットのリポジトリアクセス（個別）

【Codespaces】
  github_codespaces_secret                   - リポジトリレベルのCodespacesシークレット
  github_codespaces_organization_secret      - OrganizationレベルのCodespacesシークレット
  github_codespaces_organization_secret_repositories - Org Codespacesシークレットのリポジトリアクセス
  github_codespaces_user_secret              - ユーザーレベルのCodespacesシークレット

【Enterprise (GitHub Enterprise)】
  github_enterprise_organization             - Enterprise配下のOrganization管理
  github_enterprise_actions_permissions      - EnterpriseレベルのActions権限
  github_enterprise_actions_runner_group     - Enterpriseランナーグループ
  github_enterprise_actions_workflow_permissions - Enterpriseワークフロー権限
  github_enterprise_security_analysis_settings  - Enterpriseセキュリティ分析設定

【その他】
  github_issue                               - Issueの管理
  github_issue_label                         - Issueラベル（個別）の管理
  github_issue_labels                        - Issueラベル（一括）の管理
  github_release                             - リリースの管理
  github_project_card                        - プロジェクトカードの管理
  github_project_column                      - プロジェクトカラムの管理
  github_app_installation_repository         - GitHub Appインストールとリポジトリの関連付け（個別）
  github_app_installation_repositories       - GitHub Appインストールとリポジトリの関連付け（一括）
  github_user_gpg_key                        - ユーザーGPGキーの管理
  github_user_ssh_key                        - ユーザーSSHキーの管理
  github_user_invitation_accepter            - ユーザー招待の承認
  github_emu_group_mapping                   - EMU（Enterprise Managed Users）グループマッピング
